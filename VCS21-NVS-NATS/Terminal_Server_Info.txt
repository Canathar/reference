   Login/Logout Of The Terminal Server
   ═══════════════════════════════════
      1. Open a terminal window as a non-root user
      2. Login to the terminal server by typing the following command(s) into the terminal window:

            export TERM=vt100
            telnet <terminal-server-ip-address>
            (ex: telnet 192.168.155.242)

      3. Enter the login name for the desired session (ex: admin)
      4. Enter the password for the login (ex for admin: superuser)
      5. When ready to logout, type the following command(s) into the terminal window:

            logout



   Graphical Terminal Server Configuration
   ═══════════════════════════════════════
      1. Login to the terminal server as an admin user (procedure shown above)
      2. Display the graphical menu by typing the following command(s) into the terminal window:

            screen
            (NOTE: On later model terminal servers, the "menu" command performs the same function)



   Display The Server Configuration Of The Terminal Server
   ═══════════════════════════════════════════════════════
      1. Login to the terminal server as either an admin or normal user (procedure shown above)
      2. Display the server configuration by typing the following command(s) into the terminal window:

            show server



   Display The Status Of The Serial Lines In The Terminal Server
   ═════════════════════════════════════════════════════════════
      1. Login to the terminal server as an admin user (procedure shown above)
      2. Display the satus of the serial lines by typing the following command(s) into the terminal window:

            show line *

   Access Line Quick Menu
   ══════════════════════
      1. Login to the terminal server as an admin user (procedure shown above)
      2. Display the line quick menu by typing the following command(s) into the terminal window:

            line

            NOTE: ~ is the default menu hotkey



   Access IOLAN+ Configuration Menu
   ════════════════════════════════
      1. Login to the terminal server as an admin user (procedure shown above)
      2. Display the configuration menu by typing the following command(s) into the terminal window:

            iolan+

            NOTE: ~ is the default menu hotkey
